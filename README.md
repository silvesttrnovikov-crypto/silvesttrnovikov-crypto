Назначение 
REST-сервис на FastAPI для управления проектами и задачами, содержащий интеллектуальный модуль автоматического распределения задач между разработчиками на основе их текущей часовой загрузки и квалификации (алгоритм балансировки ресурсов).

Как запустить локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `uvicorn app.main:app --reload`
3. Откройте интерактивную документацию API: `http://127.0.0.1:8000/docs`
